1. Decision Flowchart (Diamonds + YES/NO Arrows)

Used in: S9 When To Use, S12 Bug root cause diagrams

Diamond Connection Math

Diamond at translate(cx, cy) with points="0,-42 90,0 0,42 -90,0"

  Connection points (ABSOLUTE coordinates):
    TOP:    (cx, cy - 42)
    BOTTOM: (cx, cy + 42)
    LEFT:   (cx - 90, cy)
    RIGHT:  (cx + 90, cy)

  Line from LEFT tip going left:
    x1 = cx - 90    ← starts at diamond edge
    y1 = cy
    x2 = target_x   ← ends at box edge
    y2 = cy

  Line from BOTTOM tip going down:
    x1 = cx
    y1 = cy + 42     ← starts at diamond edge
    x2 = cx
    y2 = next_cy - 42 ← ends at next diamond top (leave gap for arrow marker)

  WRONG: x1="cx" (center) — line starts INSIDE the diamond
  RIGHT: x1="cx - 90" (left tip) — line starts at the edge

2. UML Class Diagram (Boxes + Inheritance/Composition)

Used in: S4 Core UML, S10 Comparisons

Box Connection Math

Rectangle at (x, y) with width=W, height=H:

  Connection points:
    TOP-CENTER:    (x + W/2, y)
    BOTTOM-CENTER: (x + W/2, y + H)
    LEFT-CENTER:   (x, y + H/2)
    RIGHT-CENTER:  (x + W, y + H/2)

  Inheritance arrow from child to parent:
    x1 = child_x + child_W/2     ← child top center
    y1 = child_y                  ← child top edge
    x2 = parent_x + parent_W/2   ← parent bottom center
    y2 = parent_y + parent_H     ← parent bottom edge

3. Execution Flow (Horizontal Pipeline)

Used in: S5 Code execution flow, S12 Bug incident flows

Pipeline Connection Math

Boxes are 120px wide with 50px gaps between them:
  Box 1: x=10,  right edge = 10 + 120 = 130
  Gap:   130 → 180 (arrow line)
  Box 2: x=180, right edge = 180 + 120 = 300
  Gap:   300 → 350 (arrow line)
  Box 3: x=350, right edge = 350 + 120 = 470

Arrow from box N to box N+1:
  x1 = boxN_x + boxN_width     ← right edge of source
  x2 = boxN+1_x - 10           ← leave 10px gap for arrow marker
  y1 = y2 = boxN_y + boxN_height/2  ← vertical center

5. Standard Defs Block (Copy Into Every SVG)

<defs>
  <!-- Arrow marker — ALWAYS scope ID with prefix -->
  <marker id="PREFIX-arr" markerWidth="10" markerHeight="8" refX="10" refY="4" orient="auto">
    <path d="M0,0 L10,4 L0,8" fill="#14b8a6"/>
  </marker>
  <!-- Red variant for NO/error paths -->
  <marker id="PREFIX-arr-red" markerWidth="10" markerHeight="8" refX="10" refY="4" orient="auto">
    <path d="M0,0 L10,4 L0,8" fill="#ef4444"/>
  </marker>
  <!-- Open triangle for inheritance -->
  <marker id="PREFIX-inherit" markerWidth="12" markerHeight="10" refX="12" refY="5" orient="auto">
    <polygon points="0,0 12,5 0,10" fill="none" stroke="#14b8a6" stroke-width="1.5"/>
  </marker>
</defs>

Cheat Sheet: Constants

VIEWBOX:        0 0 760 {height}   (max 760 wide for sidebar pages)
STROKE-WIDTH:   2 (lines/arrows), 1.5 (box borders), 1 (separators)
MARKER SIZE:    markerWidth="10" markerHeight="8"
DIAMOND RADIUS: horizontal=90, vertical=42
FONT SIZES:     12-13px (labels), 14px (titles), 10-11px (annotations)
COLORS:
  accent:   #14b8a6 (teal)    highlight boxes, YES arrows
  error:    #ef4444 (red)      NO arrows, warning boxes
  muted:    #64748b            annotations, secondary boxes
  text:     #e2e8f0 (primary)  #94a3b8 (secondary)
  bg:       #1e293b (boxes)    #334155 (borders)
  purple:   #a78bfa            comparison alt color
  blue:     #38bdf8            context/usage highlights
BOX SIZES:     120-200px wide, 50-70px tall, rx="8"
GAP:           40-50px between boxes (room for arrow + label)
